# Distributary

Distribute webhooks to multiple recipients, especially for apps that support only one webhook url.

## Installation

Add this line to your application's Gemfile:

    gem 'distributary'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install distributary
